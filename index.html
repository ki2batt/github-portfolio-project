CSS Floats

In the CSS Floats project I had to only use floats to place elements in my webpage. In order to do this I had to utilize both the float property in my CSS and apply the clearfix property in my HTML and CSS.

Bootstrap Project

For the Bootstrap project I had to essentially learn the Bootstrap framework and apply various information and elements that include my name, four descriptive columns with info based upon my favorite hobby, food, movie, and book. I had to also apply a few images and a dropdown menu linking to my favorite websites.

Responsive Photo Gallery

This particular project was based off of the CSS Floats Project. However, in this project I had to apply two media queries for the images to make it responsive. This made it all the more challenging to make it responsive when applying EMs, and REMS for the font sizes, padding, and/or margins.

Foundation Project

Similar to the Bootstrap project although this time I had to apply four columns with my information using the XY Grid system. I had to include an accordian example with three panels, and apply a drop down menu with three links to favorite websites with unique icons alongside them.
